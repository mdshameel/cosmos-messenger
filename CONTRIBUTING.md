Contributing to COSMOS
Thank you for your interest in contributing! Here's how to get started.
🐛 Reporting Bugs

Check if the issue already exists in GitHub Issues
If not, open a new issue with:

A clear title and description
Steps to reproduce the bug
Expected vs actual behavior
Browser and OS info
Screenshots if relevant



💡 Suggesting Features
Open an issue with the label enhancement and describe:

What problem does this solve?
How should it work?
Any examples or mockups?

🔧 Submitting Code

Fork the repository
Create a branch — git checkout -b feature/your-feature-name
Make your changes to index.html
Test thoroughly — open the file in a browser and verify everything works
Commit with a clear message — git commit -m "Add: short description"
Push — git push origin feature/your-feature-name
Open a Pull Request against the main branch

📋 Pull Request Guidelines

Keep PRs focused — one feature or fix per PR
Describe what your PR does and why
Test in at least Chrome and Firefox
If adding a new feature, update README.md accordingly
Make sure the app still loads and connects to Supabase correctly
